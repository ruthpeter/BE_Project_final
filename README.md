
## Quick Start
### Run with Docker(Recommended)
Not that the runtime node version of the System should match with that written inside
the dockerfile
#### Requirements
* yarn
* docker
* docker-compose
```bash
$ git clone https://github.com/ruthpeter/BE_Project_final
$ cd BE_Project_final
$ yarn
$ docker-compose up --build
```

### Run without Docker
#### Requirements
* yarn
```bash
$ git clone https://github.com/ruthpeter/BE_Project_final
$ cd BE_Project_final
$ yarn
$ yarn run start
```
